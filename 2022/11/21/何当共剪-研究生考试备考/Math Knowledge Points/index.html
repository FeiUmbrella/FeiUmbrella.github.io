Math Knowledge Points

此文章目的是记忆公式，故不做展开证明。

公式摘抄自：《考研数学-复习全书*基础篇》。

常见三角函数

倍角

  • \(\cos2\alpha = \cos^2\alpha - \sin^2\alpha\)
  • \(\sin2\alpha = 2\sin\alpha\cos\alpha\)
  • \(\tan2\alpha = \frac{2\tan\alpha}{1-\tan^2\alpha}\)

半角

  • \(\sin^2 \frac{\alpha}{2} = \frac{1 - \cos\alpha}{2}\)
  • \(\cos^2 \frac{\alpha}{2} = \frac{1 + \cos\alpha}{2}\)
  • \(\tan^2 \frac{\alpha}{2} = \frac{1 - \cos\alpha}{1 + \cos\alpha}\)
  • \(\sin \frac{\alpha}{2} = \frac{\sin\alpha}{1 + \cos\alpha} = \frac{1 - \cos\alpha}{\sin\alpha}\)

和差

  • \(\sin(\alpha \pm\beta) = \sin\alpha\cos\beta \pm\cos\alpha\sin\beta\)
  • \(\cos(\alpha \pm\beta) = \cos\alpha\cos\beta \mp\sin\alpha\sin\beta\)
  • \(\tan(\alpha \pm\beta) = \frac{\tan\alpha\pm\tan\beta}{1\mp\tan\alpha\tan\beta}\)

Other

  • \(\cot\alpha = \frac{1}{\tan\alpha}\)
  • \(\csc\alpha = \frac{1}{\sin\alpha}\)
  • \(\sec\alpha = \frac{1}{\cos\alpha}\)
  • \(1+\tan^2\alpha = \sec^2\alpha\)
  • \(1+\cot^2\alpha = \csc^2\alpha\)

等价无穷小

当\(x \rightarrow 0\)时：

常用等价无穷小

  • \(x \sim \sin x \sim \tan x \sim \arcsin x \sim \arctan x \sim e^x - 1 \sim ln(x+1)\)
  • \(1 - \cos x \sim \frac{1}{2}x^2\)
  • \((1 + x)^m - 1 \sim mx \quad [m \neq 0]\)

拓展等价无穷小

  • \(x - \sin x \sim \frac{1}{6}x^3\)
  • \(x - \arcsin x \sim -\frac{1}{6}x^3\)
  • \(x -ln(1 + x) \sim \frac{1}{2}x^2\)
  • \(x - \tan x \sim -\frac{1}{3}x^3\)
  • \(x - \arctan x \sim \frac{1}{3}x^3\)

《极限-常用等价无穷小推导》author-蜡笔不好吃

两个重要极限

第一个重要极限：\(\frac{0}{0}\)型

  • \[\lim\limits_{\alpha\to无穷小量}\frac{\sin \alpha}{\alpha} = 1\]

第二个重要极限：\(1^\infty\)型

若\(\lim\alpha(x) = 0,\lim\beta(x) = \infty\)，且\(\lim\alpha(x)\beta(x) = A\)

则：\(\lim[1+\alpha(x)]^{\beta(x)} = e^A\)

常见基本不等式

  • \(a + b \geq 2\sqrt{ab}\)；当且仅当\(a=b\)等号成立

基本不等式链

  • \(\frac{2}{\frac{1}{a} + \frac{1}{b}} \leq \sqrt{ab} \leq \frac{a + b}{2} \leq \sqrt{\frac{a^2 + b^2}{2}}\)

《从零开始的基本不等式及其应用大全——超详细梳理》author：知乎用户O87iHP

常见求导

基本初等函数求导

  • \((a^x)\prime = a^x\ln a\)
  • \((x^a)\prime = ax^{a-1}\)
  • \((\log_ax)\prime = \frac{1}{x\ln a}\)
  • \((\ln \lvert x\rvert)\prime = \frac{1}{x}\)
  • \([\ln (x + \sqrt{(x^2+1)})]\prime = \frac{1}{\sqrt{x^2+1}}\)
  • \((\tan x)\prime = \sec^2x\)
  • \((\cot x)\prime = -\csc^2x\)
  • \((\sec x)\prime = \sec x\tan x\)
  • \((\csc x)\prime = -\csc x \cot x\)
  • \((\arcsin x)\prime = \frac{1}{\sqrt{1 - x^2}}\)
  • \((\arccos x)\prime = -\frac{1}{\sqrt{1 - x^2}}\)
  • \((\arctan x)\prime = \frac{1}{1 + x^2}\)
  • \((\arccot x)\prime = -\frac{1}{1 + x^2}\)

高阶求导

类比二项式展开

  • \([u(x)v(x)]^{(n)} = \sum\limits_{i = 0}^{n}C_n^iu^{(n - i)}v^{(i)}\)

常见积分

==\(+C\)==

  • \(\int x^{\alpha}dx = \frac{1}{\alpha+1}x^{\alpha+1}+C\)
  • \(\int \frac{1}{x}dx = \ln \lvert x\rvert+C\)
  • \(\int a^xdx = \frac{a^x}{\ln a}+C\)
  • \(\int\frac{1}{a^2+x^2}dx = \frac{1}{a}\arctan \frac{x}{a} + C\)
  • \(\int\frac{1}{\sqrt{a^2-x^2}}dx = \arcsin \frac{x}{a} + C\)
  • \(\int\frac{1}{a^2-x^2}dx = -\frac{1}{2a}\ln \lvert\frac{a+x}{a-x}\rvert + C\)
  • \(\int\frac{dx}{\sqrt{x^2\pm a^2}}dx = \ln\lvert x+\sqrt{x^2\pm a^2}\rvert + C\)
  • \(\int \tan x dx = -\ln\lvert\cos x\rvert+C\)
  • \(\int \cot x dx = \ln\lvert\sin x\rvert+C\)
  • \(\int \sec x dx = -\ln\lvert\sec x + \tan x\rvert+C\)
  • \(\int \csc x dx = -\ln\lvert\csc x - \cot x\rvert+C\)
  • \(\int \sec^2 x dx = \tan x+C\)
  • \(\int \csc^2 x dx = -\cot x + C\)
